# Imprint Logistics LLC — Website

Official website for **Imprint Logistics LLC**, an authorized Bitcoin ATM (BTM) placement and distribution company representing Bitcoin Depot, RockItCoin, and Unbank.

## 🚀 Live Site
Deployed on Vercel — [View Live](#)

## 📁 Structure
```
├── index.html          # Homepage
├── css/style.css       # Global styles
├── js/main.js          # Cursor, particles, animations
├── pages/
│   ├── about.html      # About us
│   ├── services.html   # All services
│   ├── partners.html   # BTM operator partners
│   ├── team.html       # Leadership team
│   ├── faq.html        # FAQ with accordion
│   └── contact.html    # Contact form + WhatsApp
├── vercel.json         # Vercel deployment config
└── README.md
```

## ✨ Features
- Custom animated cursor with glow trail
- Particle network background
- Scroll-reveal animations
- Animated counters
- FAQ accordion
- Responsive mobile menu
- WhatsApp floating button
- Clean URLs via Vercel

## 🛠 Deploy
1. Push to GitHub
2. Import in [Vercel](https://vercel.com)
3. Deploy — done!
